    - Export `BlueprintWithCandidates` interface: `{ blueprint: GenerationBlueprint, candidates: Map<string, ResolvedCandidate> }`
    - Export `async function resolveBlueprint(key: DeckStructureKey, dealContext: DealContext): Promise<BlueprintWithCandidates | null>`

    Implementation steps (from research Pattern 1):
    1. Validate key via `resolveDeckStructureKey(key.touchType, key.artifactType)` -- let it throw on invalid
    2. Query `prisma.deckStructure.findFirst({ where: { touchType, artifactType } })` -- return null if not found
    3. Parse `structureJson` as `DeckStructureOutput` -- return null if parse fails or sections empty
    4. Collect all slideIds across all sections via `flatMap` -- return null if none
    5. Batch query `prisma.slideEmbedding.findMany({ where: { id: { in: allSlideIds }, archived: false }, select: { id, templateId, classificationJson, thumbnailUrl, slideIndex, slideObjectId } })`
    6. Build `slideMap: Map<string, SlideEmbedding>`
    7. Batch query `prisma.template.findMany({ where: { id: { in: uniqueTemplateIds } }, select: { id, presentationId } })` -- separate query because NO FK relation exists
    8. Build `templateMap: Map<string, Template>`
    9. Build `candidates: Map<string, ResolvedCandidate>` from slideMap + templateMap
    10. Map `DeckSection[]` -> `SectionSlot[]`, sorted by `order`, filtering candidateSlideIds to only IDs present in slideMap
    11. Return `{ blueprint: GenerationBlueprint, candidates }` or null if all sections empty after filtering

    Import paths:
