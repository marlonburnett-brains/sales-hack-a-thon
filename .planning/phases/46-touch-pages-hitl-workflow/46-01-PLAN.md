    - Export `HitlStage` type: `"skeleton" | "lowfi" | "highfi"`
    - Export `HITL_STAGES` constant array: `[{ key: "skeleton", label: "Outline" }, { key: "lowfi", label: "Draft" }, { key: "highfi", label: "Final" }]`

    - Use `CheckCircle` icon from lucide-react for completed stages (matches existing pattern)
    - Per UI/UX skill: minimum 44x44px touch targets, cursor-pointer on clickable elements, 150-300ms transitions
