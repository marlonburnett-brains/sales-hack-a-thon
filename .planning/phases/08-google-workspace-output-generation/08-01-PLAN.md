**Create `apps/agent/src/lib/deck-assembly.ts`** with a single exported function `createSlidesDeckFromJSON`:

```typescript
export interface DeckFromJSONParams {
  slideJSON: SlideAssembly;
  companyName: string;
  primaryPillar: string;
  dealFolderId: string;
}

export interface DeckFromJSONResult {
  presentationId: string;
  deckUrl: string;
  slideCount: number;
}

export async function createSlidesDeckFromJSON(params: DeckFromJSONParams): Promise<DeckFromJSONResult>
```

Implementation logic:

1. **Copy branded template** via `getDriveClient().files.copy()` with:
   - `fileId: env.GOOGLE_TEMPLATE_PRESENTATION_ID`
   - `name: "${companyName} - ${primaryPillar} - ${YYYY-MM-DD}"`
   - `parents: [dealFolderId]`
   - `supportsAllDrives: true`

2. **Read the copied presentation** via `getSlidesClient().presentations.get()` to discover template slide objectIds. Build a `templateSlideMap` by matching slide content patterns or using the slide index as a mapping key. The template deck is expected to have at least one generic content slide. Map section types to template slides:
   - If a slide matching a section type pattern exists, map it
   - Otherwise, fall back to a generic "content" template slide for all section types
   - Store the discovered objectIds in a `Map<string, string>` (sectionType -> pageObjectId)

3. **Delete original template slides** from the copy (they'll be replaced by duplicated+customized slides). Keep track of which template objectIds exist for duplication.

4. **For each slide in the SlideJSON** (in order):
   a. Determine the template slide to duplicate based on `sectionType`
   b. Use `duplicateObject` batchUpdate request to create a new slide from the template
   c. Read the duplicate's new objectId from the batchUpdate response (`replies[0].duplicateObject.objectId`)
   d. Build `replaceAllText` requests with `pageObjectIds: [newSlideObjectId]` to inject:
      - `{{slide-title}}` -> `slide.slideTitle`
      - `{{bullet-content}}` -> `slide.bullets.join('\n')`
      - `{{speaker-notes}}` -> inject via `insertText` on the notes page element (or use a `{{speaker-notes}}` placeholder if template supports it)
   e. Execute the replacement batchUpdate

   **For retrieved slides (sourceType="retrieved"):** Attempt to copy the source slide from the original presentation:
   - Try `drive.files.copy({ fileId: slide.presentationId })` if presentationId available from sourceBlockRef lookup
   - If source copy succeeds, extract the target slide and inject bespoke copy
   - If source copy fails (403/404/missing presentationId), fall back to the branded template approach (same as synthesized slides)
   - Log a warning on fallback: `[deck-assembly] Source unavailable for ${sourceBlockRef}, using branded template fallback`

   **For synthesized slides (sourceType="synthesized"):** Always use branded template duplicate + text replacement (no source to copy from).

5. **Clean up template slides** that were used as duplication sources but should not appear in the final deck. Delete all original template slide objectIds that were not duplicated into the final order.

6. **Make publicly viewable** via `makePubliclyViewable(presentationId)`.

7. Return `{ presentationId, deckUrl, slideCount }`.

**Critical constraints:**
- ALWAYS re-read presentation via `presentations.get()` after ANY batchUpdate mutation to get current objectIds (objectId drift after copy/duplicate)
- ALWAYS scope `replaceAllText` with `pageObjectIds: [targetSlideObjectId]` to prevent cross-slide contamination
- Use `supportsAllDrives: true` on all Drive API calls
- Handle Google API errors with try/catch per slide (one slide failure should not crash the entire deck)

5. `makePubliclyViewable(documentId)`
6. Return `{ documentId, docUrl: "https://docs.google.com/document/d/${documentId}/edit" }`

2. **Create the step:**
   ```typescript
   const createSlidesDeck = createStep({
     id: "create-slides-deck",
     inputSchema: z.object({
       interactionId: z.string(),
       briefId: z.string(),
       slideJSON: z.string(),
       slideCount: z.number(),
       retrievalSummary: z.string(),
     }),
     outputSchema: z.object({
       interactionId: z.string(),
       briefId: z.string(),
       slideJSON: z.string(),
       slideCount: z.number(),
       deckUrl: z.string(),
       dealFolderId: z.string(),
     }),
     execute: async ({ inputData }) => {
       // a. Fetch brief + deal + company for naming and folder
       const brief = await prisma.brief.findUniqueOrThrow({
         where: { id: inputData.briefId },
         include: {
           interaction: {
             include: { deal: { include: { company: true } } },
           },
         },
       });

       const deal = brief.interaction.deal;
       const company = deal.company;
       const primaryPillar = brief.primaryPillar;
       const dateStr = new Date().toISOString().split("T")[0];

       // b. Get or create per-deal Drive folder
       const dealFolderId = await getOrCreateDealFolder({
         companyName: company.name,
         dealName: deal.name,
         parentFolderId: env.GOOGLE_DRIVE_FOLDER_ID,
       });

       // Update deal's driveFolderId if not set
       if (!deal.driveFolderId) {
         await prisma.deal.update({
           where: { id: deal.id },
           data: { driveFolderId: dealFolderId },
         });
       }

       // c. Deserialize SlideJSON
       const slideAssembly = JSON.parse(inputData.slideJSON);

       // d. Create the deck
       const result = await createSlidesDeckFromJSON({
         slideJSON: slideAssembly,
         companyName: company.name,
         primaryPillar,
         dealFolderId,
       });

       console.log(`[create-slides-deck] Created deck: ${result.deckUrl} (${result.slideCount} slides)`);

       return {
         interactionId: inputData.interactionId,
         briefId: inputData.briefId,
         slideJSON: inputData.slideJSON,
         slideCount: result.slideCount,
         deckUrl: result.deckUrl,
         dealFolderId,
       };
     },
   });
   ```
