<objective>
Build the pre-call briefing backend pipeline: a 6-step Mastra workflow that takes company name, industry, buyer role, and meeting context, then researches the company via Gemini, queries AtlusAI for case studies, generates role-specific hypotheses and discovery questions, creates a formatted Google Doc in the per-deal Drive folder, and records the interaction. Wire the workflow into the Mastra instance and create the API client + server action plumbing.

Purpose: The backend pipeline is the core intelligence of the pre-call briefing flow. Without it, the UI has nothing to call.
Output: Working workflow registered in Mastra, callable via API, producing Google Docs and InteractionRecords.
</objective>

    ```typescript
    // Pre-Call Briefing Actions
    export async function startPreCallWorkflow(
      dealId: string,
      formData: {
        companyName: string;
        industry: string;
        buyerRole: string;
        meetingContext: string;
      }
    ): Promise<WorkflowStartResult> {
      return fetchJSON<WorkflowStartResult>(
        "/api/workflows/pre-call-workflow/start",
        {
          method: "POST",
          body: JSON.stringify({ inputData: { dealId, ...formData } }),
        }
      );
    }

    export async function getPreCallWorkflowStatus(
      runId: string
    ): Promise<WorkflowRunResult> {
      return fetchJSON<WorkflowRunResult>(
        `/api/workflows/pre-call-workflow/${runId}`
      );
    }
    ```

    **Step 2: Add server actions.**
    In `apps/web/src/lib/actions/touch-actions.ts`:
    - Import the two new functions: `startPreCallWorkflow`, `getPreCallWorkflowStatus`
    - Add to the import type: no new types needed (WorkflowStartResult and WorkflowRunResult already imported)
    - Add two server actions:

    ```typescript
    // Pre-Call Briefing Actions
    export async function generatePreCallBriefingAction(
      dealId: string,
      formData: {
        companyName: string;
        industry: string;
        buyerRole: string;
        meetingContext: string;
      }
    ) {
      const result = await startPreCallWorkflow(dealId, formData);
      revalidatePath(`/deals/${dealId}`);
      return result;
    }

    export async function checkPreCallStatusAction(runId: string) {
      return getPreCallWorkflowStatus(runId);
    }
    ```
