## Bug Found

### Confidence Score 100x Inflation

**Severity:** Medium (cosmetic, does not block functionality)
**Files:**
- `apps/web/src/components/slide-viewer/classification-panel.tsx` line 105
- `apps/web/src/app/(authenticated)/slides/slide-library-client.tsx` line 242

Both multiply `slide.confidence` by 100, but the value is already stored as 0-100. A slide with 75% confidence displays as "7500%". Fix: use `Math.round(slide.confidence)` instead of `Math.round(slide.confidence * 100)`.
