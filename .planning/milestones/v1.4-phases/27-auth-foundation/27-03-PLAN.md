2. **Add POST /atlus/detect route** for on-demand triggering (e.g., after user manually takes action on an ActionRequired item):
   ```typescript
   registerApiRoute("/atlus/detect", {
     method: "POST",
     handler: async (c) => {
       const body = await c.req.json();
       const data = z.object({
         userId: z.string().min(1),
         email: z.string().email(),
         googleAccessToken: z.string().min(1),
       }).parse(body);

     },
   ```
