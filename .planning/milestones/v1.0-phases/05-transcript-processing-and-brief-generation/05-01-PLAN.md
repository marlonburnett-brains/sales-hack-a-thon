   - Props: dealId, companyName, industry, onClose
   - States: "input" | "extracting" | "fieldReview" | "generating" | "briefResult"
   - Input state:
     - Pre-filled company name and industry (read-only display, same as Touch 1)
     - Industry select (shadcn Select, pre-selected from deal but editable, populated from INDUSTRIES constant)
     - Cascading subsector select (shadcn Select, options filtered from SUBSECTORS[selectedIndustry]). Reset subsector when industry changes.
     - Large freeform Textarea for transcript paste (placeholder: "Paste your meeting transcript here...")

   - Extracting state: GenerationProgress with "Extracting fields from transcript..."

   - Generating state: GenerationProgress with step-by-step messages ("Mapping solution pillars...", "Generating brief...", "Framing ROI outcomes...")
