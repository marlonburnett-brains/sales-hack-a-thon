  <action>
Create an `apps/tutorials/src/remotion/effects/` directory and add focused, reusable Remotion components.

**ZoomPan.tsx**
- Export `ZoomPan` component that wraps children in an `AbsoluteFill` or styled div and applies CSS transforms.
- Use `useCurrentFrame()`, `useVideoConfig()`, `interpolate()`, and Remotion easing so the motion follows the locked behavior:
  - zoom in at step start
  - hold through narration
  - zoom out during the last 0.5 seconds (15 frames at 30fps)
- When no target/scale is provided, render children unchanged.
- Support configurable transform origin from coordinate props.

**StepBadge.tsx**
- Render a top-left pill badge in the form `Step {current} of {total}`.
- Use dark semi-transparent background, white text, rounded corners, subtle shadow.
- Add a short fade-in.

**Callout.tsx**
- Render a rounded label with annotation text and a subtle arrow pointer.
- Support target coordinate props.
- Animate in with fade + slight upward slide over ~9 frames.
- Only implement the component; conditional rendering happens in Plan 02 wiring.

**ShortcutBadge.tsx**
- Render a compact keyboard shortcut badge for values like `Cmd+S` or `⌘+S`.
- Keep the styling visually aligned with StepBadge.

**AnimatedCursor.tsx**
- Render a macOS-style arrow cursor using CSS or inline SVG. Avoid adding a binary asset if a simple inline SVG keeps the component self-contained.
- Animate movement from a `from` point to a `to` point using `interpolate()` and easing for a smooth human-like arc.
- Include an optional click ripple ring that expands and fades over ~0.3 seconds after arrival.
- If coordinates are absent, render nothing.

**IntroSlate.tsx**
- Render a 3-second intro slate using title, description, and text-based AtlusDeck branding.
- Fade in from black.
- Do not depend on a not-yet-existing logo asset; use a text wordmark fallback unless a real asset is already present.

**OutroSlate.tsx**
- Render a 4-second completion slate with AtlusDeck wordmark, `Tutorial Complete`, and optional `Next: {tutorial}` copy.
- Fade to black on exit.

General implementation rules:
- Keep all components pure and prop-driven.
- No composition wiring yet; Plan 02 handles integration.
- Prefer CSS/HTML primitives plus Remotion interpolation over canvas or SVG animation libraries.
- Use ASCII-only source.
