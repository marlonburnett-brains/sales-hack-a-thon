**In apps/agent/src/mastra/index.ts**, add 4 new API routes to the `apiRoutes` array:

1. **GET /templates** -- List all templates ordered by createdAt desc. Return array of Template objects.

2. **POST /templates** -- Create template with Drive access check:
   - Parse body with zod: `{ name: string (min 1), googleSlidesUrl: string (url), presentationId: string (min 1), touchTypes: string[] (min 1) }`
   - Call `getDriveClient().files.get({ fileId: presentationId, fields: "id,modifiedTime", supportsAllDrives: true })`
   - If accessible: set accessStatus="accessible", capture modifiedTime as sourceModifiedAt
   - If 403/404 error: set accessStatus="not_accessible", extract client_email from `JSON.parse(env.GOOGLE_SERVICE_ACCOUNT_KEY)`
   - Create template via `prisma.template.create()` with touchTypes stored as `JSON.stringify(data.touchTypes)`
   - Return `{ template, serviceAccountEmail }` (serviceAccountEmail is null when accessible)

3. **DELETE /templates/:id** -- Delete template by id. Return `{ success: true }`.

4. **POST /templates/:id/check-staleness** -- Check staleness for a single template:
   - Fetch template from DB
   - Call Drive files.get for modifiedTime (with supportsAllDrives: true)
   - If accessible: compare Drive modifiedTime to lastIngestedAt. isStale = lastIngestedAt exists AND driveModified > lastIngestedAt. Update accessStatus to "accessible" and sourceModifiedAt.
   - If error: update accessStatus to "not_accessible", return serviceAccountEmail from creds
   - Return `{ isStale, modifiedTime }` or `{ isStale: false, accessError: true, serviceAccountEmail }`

**In apps/web/src/lib/api-client.ts**, add a new "Template" section after the existing sections:

```typescript
// ────────────────────────────────────────────────────────────
// Templates
// ────────────────────────────────────────────────────────────

export interface Template {
  id: string;
  name: string;
  presentationId: string;
  googleSlidesUrl: string;
  touchTypes: string; // JSON array string
  accessStatus: string;
  lastIngestedAt: string | null;
  sourceModifiedAt: string | null;
  slideCount: number;
  createdAt: string;
  updatedAt: string;
}

export interface CreateTemplateResult {
  template: Template;
  serviceAccountEmail: string | null;
}

export interface StalenessCheckResult {
  isStale: boolean;
  modifiedTime?: string;
  accessError?: boolean;
  serviceAccountEmail?: string;
}

export async function listTemplates(): Promise<Template[]> {
  return fetchJSON<Template[]>("/templates");
}

export async function createTemplate(data: {
  name: string;
  googleSlidesUrl: string;
  presentationId: string;
  touchTypes: string[];
}): Promise<CreateTemplateResult> {
  return fetchJSON<CreateTemplateResult>("/templates", {
    method: "POST",
    body: JSON.stringify(data),
  });
}

export async function deleteTemplate(id: string): Promise<{ success: boolean }> {
  return fetchJSON<{ success: boolean }>(`/templates/${id}`, {
    method: "DELETE",
  });
}

export async function checkTemplateStaleness(id: string): Promise<StalenessCheckResult> {
  return fetchJSON<StalenessCheckResult>(`/templates/${id}/check-staleness`, {
    method: "POST",
  });
}
```

Follow the exact patterns from existing Deal/Company API code. Use the same fetchJSON wrapper, same error handling.
